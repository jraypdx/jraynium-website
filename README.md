jraynium.com

Code for my single page minimal music artist website that I run from a home server.  Built using Claude with a few small edits by hand.  The site has sections for:
 - Links to social media and music streaming platforms
 - A music player to play or download my songs, as well as embedded Spotify and SoundCloud players
 - A downloads section that I use to easily send files to people by copying them over to my server and sending them a download link.  Also includes an upload button so that someone can send me files - uploads to my server and triggers an email notification
 - A contact section with a form that sends me an email

If anyone stumbles on this and wants to use this as a template to run your own music portfolio type website please feel free :)
